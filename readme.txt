0、网页
1、上传带透明的图片，实现对图片的切割

需要有2个功能，图片一般时代透明的。
1、自动识别图片，并且对图片进行分割（显示分割框），且用户手动可以增加、减少、调整分割框大小、数量--然后进行分割和下载
2、按图片百分比进行横竖设置0～多条分割线，并且可以手动调整分割线位置，

都需要可以实时预览和编辑

通过 File API 实现上传，Canvas 实现图像处理，可以使用 Fabric.js 或 Konva.js 等库简化 canvas 交互
